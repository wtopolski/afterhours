Koog AI Agents

Kotlin samples exploring the Koog AI agents framework with local LLMs — tool calling, structured output, custom strategies, and external API integration via Ollama and LM Studio.

Overview

What this is

A set of self-contained Kotlin samples that progressively explore the Koog agent framework — from a minimal "hello world" agent to a full tool-calling agent that queries a live bike-sharing API and returns structured data.

Architecture

How it fits together

All samples follow the same Koog pattern: define a strategy (node graph with edges), configure the agent (prompt + model), register tools, and run.

// Shared model config — switch once, applies everywhere
val ollama_model = gpt_oss_20b

// Define strategy as a node graph
val strategy = strategy("name") {
    val sendInput by nodeLLMRequest()
    val executeTool by nodeExecuteTool()
    val sendResult by nodeLLMSendToolResult()
    edge(nodeStart forwardTo sendInput)
    edge(sendInput forwardTo executeTool)
    // ...
}

// Create and run agent
val agent = AIAgent(
    promptExecutor = simpleOllamaAIExecutor(),
    strategy = strategy,
    agentConfig = config,
    toolRegistry = registry
)
agent.run("user prompt")
  • common/utils.kt — Shared Ollama model definitions and event handler. Change ollama_model to switch the active model globally.
  • Tool<Args, Result> — Custom tools define a ToolDescriptor (name, description, parameters) and an execute() method.
  • JsonStructuredData — Generates JSON Schema from a Kotlin @Serializable class and coerces LLM output into typed objects.
  • EventHandler — Lifecycle hooks (onToolCall, onAfterLLMCall, onAgentFinished) for logging and debugging.
  • LMStudioClient — Shows how to swap the LLM backend by implementing the LLMClient interface with Ktor HTTP calls.
Setup

Getting started

Ollama

Install from ollama.com, then pull models:

ollama pull llama3.2
ollama pull gpt-oss:20b

LM Studio

Optional. Install from lmstudio.ai, load a model, start the API server.

Only needed for lmstudio sample
# Build
./gradlew build

# Run a specific sample
./gradlew run -PmainClass=org.example.basic.MainKt
./gradlew run -PmainClass=org.example.calc.MainKt
./gradlew run -PmainClass=org.example.miejskibike.MainKt
